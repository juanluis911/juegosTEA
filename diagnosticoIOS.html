📱 Instrucciones para iOS (iPhone/iPad)

  1. Abre Safari (no Chrome ni otros navegadores)
  2. Ve a tu sitio web JuegoTEA
  3. Toca el botón "Compartir" 📤 (parte inferior de la pantalla)
  4. Busca "Añadir a pantalla de inicio" 📱
  5. Toca "Añadir" para confirmar
  6. ¡Listo! La app aparecerá en tu pantalla de inicio

🤖 Instrucciones para Android

  1. Abre Chrome o tu navegador preferido
  2. Ve a tu sitio web JuegoTEA
  3. Toca el menú (⋮) en la esquina superior derecha
  4. Selecciona "Añadir a pantalla de inicio"
  5. Toca "Añadir" para confirmar

⚠️ Importante para iOS

Safari es obligatorio: iOS solo permite instalar PWAs desde Safari. Chrome, Firefox y otros navegadores no funcionarán para la instalación.
